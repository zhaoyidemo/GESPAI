# 洛谷难度标准对照表

## 洛谷难度等级（从易到难）
1. **入门** - 最基础的题目，适合刚学编程的学生
2. **普及-** - NOIP普及组入门级别
3. **普及/提高-** - NOIP普及组中等/提高组入门
4. **普及+/提高** - NOIP普及组较难/提高组中等
5. **提高+/省选-** - NOIP提高组较难/省选入门
6. **省选/NOI-** - 省选中等/NOI入门
7. **NOI/NOI+/CTSC** - 国家队级别

## GESP 级别与洛谷难度对应关系

### GESP 1级
- 主要难度：**入门**
- 少数难题：**普及-**

### GESP 2级
- 主要难度：**入门** / **普及-**
- 以普及-为主

### GESP 3级
- 主要难度：**普及-**
- 部分难题：**普及/提高-**

### GESP 4级
- 主要难度：**普及-** / **普及/提高-**
- 少数难题：**普及+/提高**

### GESP 5级
- 主要难度：**普及/提高-** / **普及+/提高**
- 以普及+/提高为主

### GESP 6级
- 主要难度：**普及+/提高** / **提高+/省选-**
- 以提高+/省选-为主

### GESP 7级
- 主要难度：**提高+/省选-**
- 部分难题：**省选/NOI-**

### GESP 8级
- 主要难度：**提高+/省选-** / **省选/NOI-**
- 极少数超难题：**NOI/NOI+/CTSC**

## 难度替换规则

原标签 → 新标签（按 GESP 级别）：

### GESP 1级
- `easy` → `入门`

### GESP 2级
- `easy` → `普及-`

### GESP 3级
- `easy` → `普及-`
- `medium` → `普及/提高-`

### GESP 4级
- `easy` → `普及-`
- `medium` → `普及/提高-`
- `hard` → `普及+/提高`

### GESP 5级
- `easy` → `普及/提高-`
- `medium` → `普及+/提高`
- `hard` → `提高+/省选-`

### GESP 6级
- `easy` → `普及+/提高`
- `medium` → `提高+/省选-`
- `hard` → `提高+/省选-`

### GESP 7级
- `easy` → `普及+/提高`
- `medium` → `提高+/省选-`
- `hard` → `省选/NOI-`

### GESP 8级
- `easy` → `提高+/省选-`
- `medium` → `省选/NOI-`
- `hard` → `省选/NOI-`
